# Arbitrage01-3 新電腦部署指南

## 初始設定（新電腦）：
```bash
cd /path/to/Arbitrage01-3
./setup_environment.sh
```

## 日常使用（二選一）：

### 方式A：手動
```bash
source venv/bin/activate
python3 master_controller_v3.py --exchanges binance bybit --top_n 5 --start_date 2025-07-14 --end_date 2025-07-15 --strategy original --yes
```

### 方式B：腳本
```bash
./start_master.sh --exchanges binance bybit --top_n 5 --start_date 2025-07-14 --end_date 2025-07-15 --strategy original --yes
```
